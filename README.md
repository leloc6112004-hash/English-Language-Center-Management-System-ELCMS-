# English Language Center Management System (ELCMS)

## Overview
This project is a Business Analysis case study for an English Language Center Management System.

The system helps manage students, courses, schedules, and operations of an English center.

---

## Objectives
- Analyze business requirements
- Design system features
- Improve management efficiency

---

## Stakeholders
- Admin: manage system and reports
- Staff: manage students and courses
- Students: register and study

---

## Core Features

### User Management
- Create / update / delete users
- Role-based access

### Student Management
- Manage student information
- Track learning progress

### Course Management
- Create courses
- Assign teachers

### Schedule Management
- Manage class schedules
- Track attendance

### Payment Management
- Manage tuition fees

### Reporting
- Student reports
- Revenue reports

---

## Business Analysis

### Example User Story
As an admin, I want to manage student accounts so that I can control system access.

---

## System Design
- Frontend: Web
- Backend: API
- Database: SQL
